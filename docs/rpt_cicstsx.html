Extended Temporary Storage Summary workspace

The Extended Temporary Storage Summary workspace monitors the CICS® response to temporary-storage constraints, such as when applications are failing due to TS-related errors.

The Extended Temporary Storage Summary workspace is linked to from the Temporary Storage Summary workspace.

This workspace includes these views:
  • The Share Pool Status bar chart shows defined and connected shared pools.
  • The Auxiliary Resource Use bar chart shows the percentage of shared control intervals, I/O counts, and TS statistics.
