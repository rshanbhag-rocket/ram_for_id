System Initialization Table workspace

The predefined System Initialization Table workspace contains data on all of the keyword settings that are used to control the operation of your CICSplex. SIT keywords, their descriptions, and associated values are displayed. You can use this workspace to quickly verify that all of your SIT keywords are properly defined.
