Transaction Storage Violations workspace

The predefined Transaction Storage Violations workspace shows data on the total number of storage violations that have occurred for each transaction and the number of storage violations per transaction.

The predefined Transaction Storage Violations workspace contains the following views:

  • The Transaction Storage Violations table view
  • The Storage Violation Distribution bar chart
Note: When a storage violation has occurred, collection for this attribute group involves scanning the Program Control Table (PCT), which can carry considerable overhead. Exercise caution when using this attribute table for either workspaces or situations.
