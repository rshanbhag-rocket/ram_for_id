Units of Work workspace

The predefined Units of Work workspace lists the units of work running in the CICS® region for the selected transaction.

This workspace contains the following views:
  • The Unit of Work by Region view summarizes the activity within the CICS region for the selected unit of work, including the number of active and completed transactions associated with it.
  • The Unit of Work by Transaction view provides detail information about each associated transaction, such as CPU time and wait times.

These views are populated with attribute values from the CICSplex Units of Work attribute group.

If trace details exist for a specific transaction, this workspace can have a link to the Application Trace secondary workspace.
