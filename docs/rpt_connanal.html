Connections Analysis workspace

The predefined Connections Analysis workspace contains information on link utilization activity and workloads across terminal owning regions (TORs).

This workspace contains the following views:

  • Link Utilization bar chart, which shows the percentages of total, secondary, and primary links in use
  • Connections Analysis table view, which shows activity and workloads across terminal owning regions (TORs)

The Connections Analysis table view helps you analyze the efficiency of multiregion operation (MRO) and intersystem communication (ISC) links and detect capacity constraints that might cause bottlenecks. For example, you can readily determine the following items:

  • The number of connections between a selected region and other regions
  • The number of transactions per minute
  • The number of active LU2 connections
  • CPU usage
  • Link usage
  • The balance of work across Terminal Owning Regions (TORs)
  • The number of automatic initiate descriptors (AIDs) associated with the worst performing MRO and ISC connections

The table view also provides a link to the Link Summary workspace.
