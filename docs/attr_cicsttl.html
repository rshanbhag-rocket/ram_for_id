CICSplex Task Time Summary attribute group

The Transaction times for a task report where the time was spent during the life of this transaction, based on data collected by the CICS Monitoring Facility.

CICS Region Name Indicates the name that identifies a CICS region. The format is an eight-character string.

Origin Node The combination of MVS System ID (SMFID) and CICS region name. The format is a string.

Period Count Indicates the number of measurement periods during which the Time Spent value was accumulated in the specified category.

System ID The four-character name that uniquely identifies an active MVS operating system within a given CICSplex.

Time Category Name of the transaction monitoring field reported by CICS monitoring facility.

Time Percentage Indicates the percentage of the overall task's response time where task spent in the specified category.

Time Spent Indicates the amount of time this transaction spent in the specified category.

Time Type Indicates type of the specified category.
