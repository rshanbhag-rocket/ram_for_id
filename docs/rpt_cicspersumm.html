CICS Performance Summary workspace

The predefined CICS® Performance Summary workspace is located at the CICS node level and provides key summary information for each monitored CICS region and displays an overview of the maximum tasks percentage, transaction rate, and the amount of CPU usage. This workspace is populated from the CICSplex Region Overview and CICSplex Service Class Analysis attribute groups.

This predefined workspace contains the following views:

  • CICS Performance Summary
  • Service Level Summary table view shows the performance of all service classes for all CICS regions in the LPAR and has a link to the Service Class Summary workspace
