Application Bundle situation

The single predefined situation in this category alerts you to the efficiency status of Application Bundle resources in your CICS regions.

CICSplex_Bundle_Warning
The Application Bundle status activity.

The Application Bundle status activity identified in the alert is in a DISABLED state, indicating that the Application Bundle that has been installed in your target CICS region is not active.

Formula: If the value of CICSplex_Application_Bundle_Status is equal to Disabled then situation CICSplex_Bundle_Warning is true.

See also the CICSplex Application Bundle Summary attribute group.
