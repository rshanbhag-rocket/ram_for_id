Subpool Details workspace

The Subpool Details workspace is provided to aid you in all aspects of storage management.

Use the Subpool Details workspace in conjunction with the Storage Analysis workspace, the Dynamic Storage Detail workspace, and the Pagepool Summary workspace.

The predefined Subpool Details workspace contains the following views:

  • The Subpool allocation request stacked bar chart compares the GETMAIN with the FREEMAIN requests for each subpool.
  • The Subpools Details table view contains the DSA name, index, fixed length, boundary, access type, and location of each subpool. In addition, the number of GETMAIN and FREEMAIN requests for each subpool is reported. Also, the current page storage, the name of a specific DSA, EDSA, or GDSA, subpool type (domain or task), the unique number of the task, element count, and high water mark (in both KB and MB).
