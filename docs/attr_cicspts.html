CICSplex Previous Task Details attribute group

The CICSplex Previous Task Details attribute group provides Originating and Previous Task details within the same region or Previous Hop data from another CICS region.

CICS Region Name The name that identifies a CICS region. Each CICS region in a z/OS operating system has a unique name. This name is an alphanumeric string, with a maximum of eight characters, and case sensitive. CICS region names are always in uppercase characters.

Origin Node The combination of MVS System ID (SMFID) and the CICS region name. The value format is an alphanumeric string, maximum 32 bytes, and case sensitive.

Original APPLID The APPLID of the CICS region in which this work request (transaction) originated.

Original Network ID The network identifier from which this work request (transaction) originated.

Original Start Time The time at which the originating task was started.

Original Task Number The number of the originating task from which this work request (transaction) originated.

Original Tran ID The transaction ID (TRANID) from which this work request (transaction) originated.

Previous Hop APPLID The APPLID from the previous hop data. This is the APPLID of the CICS system of a previous task in another CICS system with which this task is associated.

Previous Hop Count The number of times there has been a request from one CICS system to another CICS system to initiate a task with which this task is associated.

Previous Hop Network ID The network identifier of the CICS system of an immediately previous task in another CICS system with which this task is associated.

Previous Hop Start Time The start time of the immediately previous task in another CICS system with which this task is associated.

Previous Hop Task Number The task number of the immediately previous task in another CICS system with which this task is associated.

Previous Hop Tran ID The transaction ID (TRANID) of the immediately previous task in another CICS system with which this task is associated.

Previous Task Count The number of times there has been a request from one task to initiate another task in the same CICS system with which this task is associated.

Previous Task Start Time The start time of the immediately previous or parent task in the same CICS system with which the task is associated.

Previous Task Number The task number of the immediately previous or parent task in the same CICS system with which the task is associated.

Previous Task Tran ID The transaction ID (TRANID) of the immediately previous or parent task in the same CICS system with which the task is associated.

System IDThe four-character name that uniquely identifies an active MVS operating system within a given CICSplex.

Task Number Indicates the task number previous task details are retrieved for.

Transaction Group Identifier Indicates transaction group identifier assigned to the task.

Transaction ID Indicates the four-character identifier for the task.
