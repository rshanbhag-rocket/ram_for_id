DB2 Summary workspace

The predefined DB2® Summary workspace contains the DB2 Summary table view, which shows whether or not a CICS® region is attached to DB2.

This table view displays data provided by the CICSplex DB2 Summary attribute group.

This workspace also contains a Take Action view that lets you enter console commands and a 3270 terminal session.

You can link to the CICS Connections workspace provided by the OMEGAMON for DB2 on z/OS product, so you can see at a glance all CICS/DB2 thread activity in the CICS region, filtered by the CICS ID.
