AID Purge

The AID Purge predefined Take Action command stops one or more CICS® Automatic Initiate Descriptors (AIDs).

Formula:

CP:AIDK ADDRESS=&CICSplex_AID_Analysis.AID_Address KILL

This command kills the AID using the AID address attribute from the CICSplex AID Analysis attribute group.

Security considerations

Your ability to use the AID purge command is determined by a separate set of security profiles that apply only to the IBM Z OMEGAMON for CICS Take Action commands.
