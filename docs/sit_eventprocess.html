Event Processing situation

The single predefined situation in this category alerts you to the efficiency status of Event Binding resources in your target CICS regions.

CICSplex_Event_Processing_Warning
The Event Binding object status activity.

The Event Binding object activity identified in the alert is in a DISABLED state, indicating that the Event Binding object definitions that have been installed in your target CICS region are not active.

Formula: If the value of Event_Processing_Status is equal to Disabled then situation CICSplex_Event_Processing_Warning is true.

See also the CICSplex Event Processing Summary attribute group.
