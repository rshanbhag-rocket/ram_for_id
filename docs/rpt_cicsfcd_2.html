File Details workspace

The File Details workspace provides detailed information about a specific file referenced by a selected document template.

The File Details workspace is linked to from the Web Services Analysis workspace's Document Template Details view.

The predefined File Details workspace contains the following views:

  • The Document Template Details table view is a repeat of the same view shown in the Web Services Analysis workspace; it is included here so you can easily link to another workspace, including the File Details workspace.

    From this view, depending on the Source Type for the table view row you select when linking, you can link to the following workspaces:

    • The Exit Details workspace, if the Source Type is EXIT
    • Back to the File Details workspace, if the Source Type is FILE, for display of information about a different file
    • The Program Details workspace, if the Source Type is PROGRAM
    • The Temporary Storage Queue Details workspace, if the Source Type is TSQ
    • The Transient Data Details workspace, if the Source Type is TDQ
  • The single-line File Control Details table view provides detailed information about a file referenced by a specific document template.
