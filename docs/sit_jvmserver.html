JVM Server Analysis situation

The single predefined situation in this category monitors the JVM server runtime environment and, in particular, the T8 TCBs, and the name of the program defined to CICS that contains the LE Runopts variables, which are used to start the JVM servers.

CICSplex_JVM_Server_Status_Warning
This situation provides a warning if the JVM Server Status is not enabled.

Formula: If the value of JVM_Server_Status is not equal to Enabled then situation CICSplex_JVM_Server_Status_Warning is true.

See also the CICSplex JVM Server Analysis attribute group.
