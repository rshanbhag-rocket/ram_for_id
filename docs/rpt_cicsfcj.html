File Control Journal and Logging workspace

The predefined File Control Journal and Logging workspace shows information on file control journals and logging. For example, the number of journal requests, forward recovery log requests and log options.

You access this workspace through the File Control Analysis workspace.

The predefined File Control Journal and Logging workspace contains the following views:

  • File Control Journal information table view: For each file it shows the number of journal requests and forward recovery log requests.
  • File Control Log Information table view: For each file it shows the z/OS® Logstream name, the log options, the Log backout option and whether the file is defined as RLS in the SIT.

There is also a link to the File Control Details workspace.
