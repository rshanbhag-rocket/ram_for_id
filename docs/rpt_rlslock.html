VSAM RLS Lock Analysis workspace

The predefined VSAM RLS Lock Analysis workspace contains the VSAM Record Level Sharing Lock Analysis table view; it provides information about tasks that are waiting for a locked VSAM record.

This information includes both suspended CICS® tasks that are waiting for this lock to be released as well as the CICS task that is holding the record, whether or not the holding task runs within the current CICS region. This data enables you to identify applications that are making poor use of resources and possibly degrading system performance by holding records for an excessive time.
