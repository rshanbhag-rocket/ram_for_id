Sample Trace Task command

The Sample Trace Task predefined Take Action command is for use in the Transactions Analysis workspace and activates a trace for a specific transaction.

Formula:

CP TRAC TASK=&CICSplex_Transaction_Analysis.Task_Number

This command starts the activation of a trace for a specific transaction by the Trace Active attribute from the CICSplex Transaction Analysis attribute group.

Security considerations

Your ability to start the Sample Trace Task command is determined by a separate set of security profiles that apply only to the IBM Z OMEGAMON for CICS Take Action commands.
