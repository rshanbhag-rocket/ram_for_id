Internet Status workspace

The predefined Internet Status workspace shows the status of the Web interface and the URL of the Web site being viewed.

The predefined Internet Status workspace contains the following views:

  • CICS® Internet Status table view that shows such data as the state of the Web interface and whether or not the TCP/IP application is waiting
  • Web browser view, shows the URL address of the Web site being viewed.
